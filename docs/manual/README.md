# 席替えアプリ（Sekigane）取扱説明書

> 教室の座席をランダムに配置し、ルーレット感覚で楽しく席替えができるアプリです。

## このアプリでできること

- 教室の「行数・列数・生徒数」を設定すると、座席をランダムに配置します。
- 生徒の名前を入力して、空いている席番号に割り当てていけます。
- 割り当てた席はハイライト表示され、誰がどこに座るか一目でわかります。
- 割り当て開始前なら、ドラッグ＆ドロップで座席の位置を入れ替えられます。
- 「リセット」でいつでも最初からやり直せます。

## 想定ユーザー・利用シーン

- 学校の先生やクラス委員など、席替えを行う方。
- 公平・ランダムに座席を決めたいとき。
- イベントや研修などで、参加者の席をランダムに決めたいとき。

## 動作環境・前提

- パソコン／スマートフォンの最近のブラウザ（Chrome / Safari など）。
- インストール不要。ブラウザだけで動きます。

## セットアップ

ふだん使う方は、公開URL（後述）をブラウザで開くだけで利用できます。自分で動かす（開発する）場合は次の手順です。

1. リポジトリを取得します。
   ```bash
   git clone https://github.com/mina-ima/Sekigane.git
   cd Sekigane
   ```
2. 依存パッケージをインストールします。
   ```bash
   npm install
   ```
3. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
   表示されたローカルURLをブラウザで開きます。

## 使い方（ステップbyステップ）

1. 最初の設定画面で「行数」「列数」「生徒数」を入力し、開始します。
2. 座席が画面にランダムに配置されます。
3. （任意）割り当てを始める前であれば、座席をドラッグ＆ドロップして位置を入れ替えられます。
4. 操作パネルで生徒の名前を入力し、空いている席番号に割り当てます。
5. 割り当てた席がハイライトされ、割り当て済みの生徒一覧に追加されます。
6. すべて割り当て終わるまで繰り返します。最初からやり直したいときは「リセット」を押します。

## 主な機能

| 機能 | 説明 |
| --- | --- |
| 座席のランダム配置 | 行数・列数・生徒数から座席をシャッフル |
| 生徒の割り当て | 名前を入力して空席番号に割り当て |
| ハイライト表示 | 割り当てた席を強調表示 |
| 座席の入れ替え | 割り当て開始前にドラッグ＆ドロップで移動 |
| リセット | 設定からやり直し |

## よくある質問・トラブルシューティング

- **Q. 座席を入れ替えられません。**
  A. 生徒の割り当てを1人でも始めると、座席の入れ替えはできなくなります（順番が崩れないようにするためです）。入れ替えたい場合はリセットしてから配置し直してください。
- **Q. 生徒数より席が多い／少ないとどうなりますか？**
  A. 「行数×列数」が席の総数です。生徒数がそれより少なければ空席ができ、席数より生徒数が多い設定はできません。
- **Q. 配置をやり直したい。**
  A. 画面右上の「リセット」を押すと、設定画面に戻ってやり直せます。

## 開発者向けメモ

- **技術スタック**: React 19 + TypeScript + Vite。スタイルは Tailwind ユーティリティクラス（クラス名はソースに直書き）。
- **主なファイル**: `App.tsx`（状態管理・シャッフルロジック）、`components/SettingsForm`（初期設定）、`components/ClassroomLayout`（座席表示・並べ替え）、`components/ControlPanel`（生徒割り当て）。エントリは `index.html` → `index.tsx`。
- **アルゴリズム**: Fisher–Yates シャッフルで座席配列をランダム化。割り当て開始後は座席移動を無効化。
- **起動コマンド**:
  ```bash
  npm install
  npm run dev        # 開発サーバー
  ```
- **ビルド/プレビュー**:
  ```bash
  npm run build      # 本番ビルド（dist/ に出力）
  npm run preview    # ビルド結果をローカル確認
  ```
- **デプロイ先**: 静的サイトとしてデプロイ可能（Vercel 等）。
- **本番URL**: 未確認（常設の公開URLはリポジトリ/デプロイ設定を参照）。
- **リポジトリ**: https://github.com/mina-ima/Sekigane
